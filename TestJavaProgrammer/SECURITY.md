# Security Policy

## Supported Versions

The following versions of this project are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to alok@example.com. All security vulnerabilities will be promptly addressed.

Please process the following:

1. Do not open an issue on GitHub for security vulnerabilities.
2. Provide a detailed description of the vulnerability.
3. Steps to reproduce the issue.
